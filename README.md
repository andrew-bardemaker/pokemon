# Pokedex

Este é um projeto de Pokedex, desenvolvido com Vue.js, onde é possível visualizar informações sobre diferentes pokémons.

## Funcionalidades

- Exibir uma lista de pokémons filtrados por nome.
- Pesquisar pokémons pelo nome.
- Exibir detalhes de um pokémon selecionado, incluindo imagem, tipos, altura, peso, movimentos e estatísticas.

## Tecnologias Utilizadas

- Vue.js: um framework JavaScript para construir interfaces de usuário.
- Vue Router: um roteador oficial para Vue.js, usado para gerenciar as rotas da aplicação.
- Vue Chart.js: uma biblioteca para criar gráficos utilizando o Chart.js no Vue.js.
- Axios: uma biblioteca para fazer requisições HTTP no navegador e no Node.js.
